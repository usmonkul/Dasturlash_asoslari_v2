## 7.1 Flexbox Nima va Nima Uchun Kerak?

### Flexbox nima?

**Flexbox** (Flexible Box Layout) - CSS ning zamonaviy layout usuli. Bu elementlarni qator yoki ustun ko'rinishida osongina joylashtirish imkonini beradi.

### Nima uchun Flexbox kerak?

#### 1. Oddiy joylashtirish
- Elementlarni bir qatorda joylashtirish
- Teng masofada tarqatish
- Markazga joylashtirish

#### 2. Muammolarni hal qilish
- Vertikal markazlash
- Teng balandlik
- Elementlar orasidagi teng masofa

### Asosiy tushunchalar

#### Flex Container
- `display: flex` xususiyatiga ega element
- Ichida flex itemlar joylashadi
- Boshqa elementlarning joylashuvini boshqaradi

#### Flex Items
- Flex container ichidagi to'g'ridan-to'g'ri farzand elementlar
- Avtomatik ravishda flex qiladi
- O'lcham va joylashuvni o'zgartirish mumkin

### Oddiy misol

```html
<style>
.flex-konteyner {
    display: flex;
    background-color: #f0f0f0;
    padding: 20px;
    gap: 10px;
}

.flex-item {
    background-color: #4caf50;
    color: white;
    padding: 20px;
    text-align: center;
    flex: 1;
}
</style>

<div class="flex-konteyner">
    <div class="flex-item">Item 1</div>
    <div class="flex-item">Item 2</div>
    <div class="flex-item">Item 3</div>
</div>
```

## 7.2 Asosiy Xususiyatlar

### Flex Container xususiyatlari

#### 1. Flex-direction - yo'nalish
```css
.flex-row {
    flex-direction: row;            /* Gorizontal (default) */
}

.flex-column {
    flex-direction: column;         /* Vertikal */
}

.flex-row-reverse {
    flex-direction: row-reverse;    /* Gorizontal teskari */
}

.flex-column-reverse {
    flex-direction: column-reverse; /* Vertikal teskari */
}
```

#### 2. Justify-content - asosiy o'q bo'yicha joylashuv
```css
.justify-start {
    justify-content: flex-start;    /* Boshidan */
}

.justify-center {
    justify-content: center;        /* Markazda */
}

.justify-end {
    justify-content: flex-end;      /* Oxiridan */
}

.justify-between {
    justify-content: space-between; /* Oralarida */
}

.justify-around {
    justify-content: space-around;  /* Atrofida */
}
```

#### 3. Align-items - kross o'q bo'yicha joylashuv
```css
.align-start {
    align-items: flex-start;        /* Yuqoridan */
}

.align-center {
    align-items: center;            /* Markazda */
}

.align-end {
    align-items: flex-end;          /* Pastdan */
}

.align-stretch {
    align-items: stretch;           /* Cho'zish (default) */
}
```

#### 4. Flex-wrap - o'rash
```css
.flex-nowrap {
    flex-wrap: nowrap;              /* O'rash yo'q (default) */
}

.flex-wrap {
    flex-wrap: wrap;                /* O'rash */
}
```

#### 5. Gap - masofa
```css
.gap-10 {
    gap: 10px;                      /* Barcha yo'nalishda 10px */
}

.gap-20 {
    gap: 20px;                      /* Barcha yo'nalishda 20px */
}

.row-gap-15 {
    row-gap: 15px;                  /* Faqat qatorlar orasida */
}

.column-gap-25 {
    column-gap: 25px;               /* Faqat ustunlar orasida */
}
```

### Flex Item xususiyatlari

#### Flex - o'lcham
```css
.flex-item {
    flex: 1;                        /* Teng o'lcham */
    flex: 2;                        /* Ikki marta katta */
    flex: 0.5;                      /* Yarim o'lcham */
}
```

### Asosiy o'q va kross o'q

#### Row yo'nalishida (flex-direction: row):
- **Asosiy o'q**: gorizontal (chapdan o'ngga)
- **Kross o'q**: vertikal (yuqoridan pastga)

#### Column yo'nalishida (flex-direction: column):
- **Asosiy o'q**: vertikal (yuqoridan pastga)
- **Kross o'q**: gorizontal (chapdan o'ngga)

### Xususiyatlarni sinab ko'rish

```html
<style>
.demo-konteyner {
    display: flex;
    background-color: #e3f2fd;
    padding: 15px;
    gap: 10px;
    margin: 15px 0;
    min-height: 80px;
}

.demo-item {
    background-color: #2196f3;
    color: white;
    padding: 15px;
    text-align: center;
    min-width: 80px;
}

/* Turli xil xususiyatlar */
.justify-center { justify-content: center; }
.justify-between { justify-content: space-between; }
.align-center { align-items: center; }
.flex-column { flex-direction: column; height: 150px; }
.flex-wrap { flex-wrap: wrap; }
.gap-20 { gap: 20px; }
.row-gap-10 { row-gap: 10px; }
.column-gap-30 { column-gap: 30px; }
</style>

<h4>Justify-content: center</h4>
<div class="demo-konteyner justify-center">
    <div class="demo-item">1</div>
    <div class="demo-item">2</div>
    <div class="demo-item">3</div>
</div>

<h4>Justify-content: space-between</h4>
<div class="demo-konteyner justify-between">
    <div class="demo-item">1</div>
    <div class="demo-item">2</div>
    <div class="demo-item">3</div>
</div>

<h4>Flex-direction: column</h4>
<div class="demo-konteyner flex-column">
    <div class="demo-item">1</div>
    <div class="demo-item">2</div>
    <div class="demo-item">3</div>
</div>

<h4>Gap: 20px</h4>
<div class="demo-konteyner gap-20">
    <div class="demo-item">1</div>
    <div class="demo-item">2</div>
    <div class="demo-item">3</div>
</div>

<h4>Row-gap: 10px, Column-gap: 30px</h4>
<div class="demo-konteyner row-gap-10 column-gap-30 flex-wrap">
    <div class="demo-item">1</div>
    <div class="demo-item">2</div>
    <div class="demo-item">3</div>
    <div class="demo-item">4</div>
</div>
```

### Flexbox xulosa

Flexbox quyidagi vazifalarni osonlashtiradi:
- **Elementlarni joylashtirish** - justify-content va align-items
- **Yo'nalishni o'zgartirish** - flex-direction
- **O'lchamlarni boshqarish** - flex xususiyati
- **O'rash** - flex-wrap
- **Masofalarni boshqarish** - gap, row-gap, column-gap
- **Markazlash** - justify-content: center + align-items: center

Bu xususiyatlar yordamida chiroyli va funksional web-sahifalar yaratish mumkin.
